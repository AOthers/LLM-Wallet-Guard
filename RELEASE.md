# LLM Wallet Guard v1.3 更新介绍

v1.3 在 DeepSeek 之外新增了「第三方中转站」平台，让 LLM Wallet Guard 从单平台工具变成真正可切换的多平台监控工具。同时大幅优化了页面布局，配置和数据展示都更紧凑。

---

## 主要更新

### 新增第三方中转站平台

平台下拉新增「第三方中转站」选项。填入 Authorization 即可查看订阅的月度额度使用情况：

- 本月已用额度
- 剩余额度
- 月度使用率百分比
- 到期时间
- 使用率 ≥ 80% 显示橙色，≥ 100% 显示红色

### 配置页动态切换

选择不同平台时，配置表单自动显示对应字段，无需重启：

| 平台 | 配置字段 |
|------|----------|
| DeepSeek | API Key、网页 Authorization、Cookie、余额阈值 |
| 第三方中转站 | Authorization、刷新间隔 |
| MiMo / OpenAI | 预留入口，提示后续支持 |

### 数据页双布局

主页面根据当前平台切换展示内容：

- **DeepSeek**：余额卡片 + 用量拆分（总消耗、输出、缓存命中、缓存未命中）
- **第三方中转站**：已用额度卡片 + 剩余额度 + 使用率 + 到期时间

### 页面大幅紧凑化

- 输入框内边距从 `7px 9px` 缩减至 `4px 8px`，高度从 30px 降至 26px
- 全局边距和间距同步收窄
- 配置页高度在第三方中转站模式下从约 440px 压缩至 210px

### 通用化命名

代码中不再包含任何具体中转站服务商名称，统一使用 `proxy` 作为内部标识，方便接入任意第三方 API 中转服务。

### 文档完善

新增 `CHANGELOG.md` 详细变更记录和 `RELEASE.md` 发布说明。README 补充第三方中转站配置示例和字段说明。

## 当前支持

### DeepSeek

- API Key 余额查询
- 网页 Authorization 完整用量
- 本月消费
- token 用量拆分（总消耗 / 输出 / 缓存命中 / 缓存未命中）
- 低余额提醒

### 第三方中转站

- 月度已用 / 剩余额度
- 使用率百分比
- 到期时间
- 超限变色提醒

### 通用功能

- 托盘常驻
- 自动刷新
- 开机自启
- 单实例运行
- 窗口拖动
